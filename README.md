# Exa-PSD: Persian Sentiment Analysis Dataset
[![DOI:10.1007/978-3-031-21438-7_60](https://zenodo.org/badge/DOI/10.1007/978-3-319-76207-4_15.svg)](https://doi.org/10.1007/s10579-025-09886-5)

This repository contains the dataset introduced in our paper: [Exa-PSD: a new Persian sentiment analysis dataset on Twitter](https://link.springer.com/article/10.1007/s10579-025-09886-5), which is designed to support research in **sentiment analysis** and **opinion mining** in Persian Language.

## Usage
You are free to use this dataset for **research and educational purposes**. If you use this dataset, please cite our paper.
